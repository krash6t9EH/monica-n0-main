Monica Tech Stack

Key Azure solutions and services used to power Monica's live agent, voice, intelligence and companion features for elderly Canadians.

Live Conversational Agent & Voice

  • Azure Communication Services (ACS)
    • Toll-free and VoIP phone lines for patient/caregiver calling
    • Video, SMS, and virtual consult routing
  • Model Router
    • Dynamic routing between Monica's conversational models (e.g., quick responses vs advanced medical reasoning)
  • Azure OpenAI Service
    • GPT-4o API for live reasoning, sentiment, dialogue and therapy
    • Supports both general and medical contexts
  • Azure AI Speech
    • Real-time speech-to-text and intelligent neural voices for Monica's agent

Orchestration & Model Management

  • Foundry IQ
    • Central management for all Monica conversational flows and model upgrades
    • Guardrails, personality tuning, and safety checks

AI Search & Patient Memory

  • Azure AI Search
    • Semantic memory and retrieval-augmented generation for patient protocols, medication reminders, and safety workflows
    • Integrates with Monica's long-term memory for patient history and care planning

Other Monica Abilities

  • Azure AI Studio
    • Model prototyping/testing, analytics dashboard, and rapid deployment
  • Azure Cognitive Services: Language
    • Intent detection, conversational understanding for Monica's wellness check-ins and emergency screening
  • Video Calling via ACS
    • Telehealth and remote check-in/video connect for family and caregivers

Agent Instructions, Tools & Accessibility

  • Instructions: To talk to Monica, use the "Talk to Monica" button on the home page, or call the dedicated toll-free number 1-800-MONICA-1.
  • Accessibility: Large fonts, screen-reader ready, simple instructions, and clear navigation for both elderly and care workers.
  • Tools and Knowledge: Monica agent is continually updated with new care protocols, medication info, and communication workflows. All conversations remain confidential and secure.
